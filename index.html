Aplicaciones de Arduino

Haz click para saber más sobre las aplicaciones de Arduino

Robótica

Arduino es excelente para la robótica, ya que tiene un gran campo de aplicación. Con Arduino puedes crear desde un pequeño robot a uno grande completamente funcional.

Sensor de Movimiento

Junto Arduino puedes crear un sensor de movimiento completamente funcional, utilizando simplemente un sensor ultrasónico y un poco de código.

Mouse o Control con Arduino

Si eres fan de los videojuegos te emocionará saber que puedes crear tu propio control con Arduino, utilizando el joystick. Este es muy fácil de utilizar e intuitivo. También puedes crear tu propio mouse, aunque este sea un poco incómodo.

Alarma

Otra gran aplicación de Arduino es la gran facilidad y efectividad al momento de crear una alarma, ya que componentes como el Buzzer o Zumbador facilitan la tarea.
