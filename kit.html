Descripción del Proyecto

Para este proyecto realizaremos un radar o sensor de movimiento, el cual armaremos junto a ustedes con los videos del kit. Utilizaremos Arduino Nano V3 en nuestro proyecto. Consideramos que este proyecto es ideal para principiantes y muy interesante, ya que puedes realizarlo de distintas formas y utilizarlo en tu vida diaria.

Si no estas familiarizado con Arduino, no te preocupes con el kit tendrás una pequeña introducción a esta innovadora y efectiva tecnología.

¿Qué incluye el kit?

Placa Arduino Nano V3 + Cable USB

Protoboard 1/2 galleta

Jumpers 10cm x10

Sensor Ultrasónico x1

Buzzer pásivo 85 dB

Set de videos para crear el proyecto

Más recursos para que sigas aprendiendo

Alarma para el auto

Una de las aplicaciones más llamativas que puedes realizar con este proyecto es una alarma para el auto. Puedes colocar el radar en la parte trasera del mismo, utilizandolo como una alarma de retroceso.

Toma en cuenta que el Buzzer es de tan solo 85 dB, por lo que no sonará tan alto.

Alarma para tu cuarto

¿Quieres saber cuando alguien se acerca a tu cuarto? Pon tu radar en la puerta de tu cuarto y mantenlo seguro.

Alarma para tu negocio

Escucha cuando un cliente entre a tu negocio sin estar en la recepción. Con la alarma puedes estar al tanto de tus clientes siendo mucho más productivo.

Medidas a tomar en cuenta antes y después de comprar y utilizar el kit

Al utilizar el kit

Maneja los componentes de Arduino con mucho cuidado.

Recuerda que siempre puedes realizar el proyecto acompañado por un adulto o amigo.

Procura que la conexiones que hagas sean iguales a como estan en los videos del kit.

Antes de comprar el kit

No nos hacemos responsables de cualquier daño que puedan sufrir los componentes de Arduino.
